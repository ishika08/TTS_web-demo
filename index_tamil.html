System Description (For Tamil Dataset)


System 1: HMM-based TTS System trained on read-speech
System 2: Neural-based autoregressive(AR) TTS System trained on read-speech (Baseline)
System 3: AR TTS system trained on synthetic E2E multi-domain speech
System 4: AR TTS system trained on synthetic HTS multi-domain speech
System 5: AR TTS system trained on synthetic E2E multi-domain speech fine-tuned on read speech
System 6: AR TTS system trained on synthetic HTS multi-domain speech fine-tuned on read speech
