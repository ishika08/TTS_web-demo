Title of the paper: E-TTS: Expressive text-to-speech synthesis for Hindi using Data Augmentation
(Communicated to SPECOM 2023)


System Description (For Hindi Dataset)


System 1: HMM-based TTS System trained on read-speech
System 2: FastSpeech2 TTS System trained on read-speech (Baseline)
System 3: FastSpeech2 trained on synthetic FS2 augmented multi-domain speech (S1)
System 4: FastSpeech2 trained on synthetic FS2 augmented multi-domain speech fine-tuned on 11 minutes of prosodically-rich speech (Proposed 1 - S2)
System 5: FastSpeech2 trained on synthetic HTS augmented multi-domain speech (H1)
System 6: FastSpeech2 trained on synthetic HTS augmented multi-domain speech fine-tuned on 11 minutes of prosodically-rich speech (Proposed 2 - H2)
